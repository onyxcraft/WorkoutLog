# Changelog

All notable changes to WorkoutLog will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-25

### Added
- Initial release of WorkoutLog
- Quick workout logging with exercise name, sets, reps, and weight
- Exercise library with 40+ exercises across 7 categories
- Custom exercise creation
- Workout templates for saving and reusing routines
- Progress charts using Swift Charts (weight and reps over time)
- Personal records tracking with PR badges
- Calendar view for workout history
- Rest timer with haptic feedback (60s, 90s, 120s, 180s presets)
- Workout summary with detailed volume statistics
- Full dark mode support
- iPad optimization
- SwiftUI + MVVM architecture
- SwiftData for local persistence
- No external dependencies
